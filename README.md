# CubeJS DuckDB Serverless

## Setup

### Created with

- **Dependencies**:

```sh
npm install -g cubejs-cli
```

- **Scaffolding**:

```sh
cubejs create cubejs-lambda-duckdb -d duckdb -t serverless
```

## Development

- **Install dependencies**:

```sh
npm install
```

- **Run development server**:

```sh
npm run dev
```
